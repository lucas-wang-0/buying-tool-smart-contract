# Get Start

## install truffle and ganache local testnet

```
npm install -g truffle
```

download ganache: http://truffleframework.com/ganache/

## install node packages

```
npm install
```

## compile

```
truffle compile
```

## deploy to testnet

```
truffle deploy
```
